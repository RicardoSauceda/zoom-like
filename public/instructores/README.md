# Fotos de Instructores

Coloca aquí las fotos de los instructores con el nombre de su **CURP** como nombre de archivo.

## Formatos soportados

- `.jpg`
- `.jpeg`
- `.png`

## Convención de nombres

```
{CURP}.jpg     ←  ideal
{CURP}.jpeg
{CURP}.png
```

## Ejemplo

Si la CURP del instructor es `GARM850101HDFLNS09`:

```
public/instructores/GARM850101HDFLNS09.jpg
```

La aplicación buscará automáticamente la imagen en este directorio al cargar la sala de reunión.  
Si no encuentra ninguna imagen, se mostrará un avatar con las iniciales del instructor.
